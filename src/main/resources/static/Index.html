***CRUD de Séries

Instruções

Projto desenvolvido utilizando o Visual Studio Code.

Para iniciar o projeto, execute a Main ou um 'Arquivo.java' dentro dos arquivos com a IDE de sua preferência, em seguida clique aqui para ser direcionado à lista de séries. Ou digite o seguinte endereço no seu navegador: localhost:8080/series/lista.

Tecnologias e dependências utilizadas

  • Bootstrap-5
  • Jquery-3

    • - Para ajudar nos efeitos e design das páginas.

  • Thymeleaf

    • - Mecanismo de templates padrão do SpringBoot.

  • Banco de dados H2

    • - Persistindo os dados em aquivo.

  • Validation

    • - Dependência de validação de dados de entrada.

  • Lombok

    • - Dependência que ajuda a simplificar a codificação.

  • Spring Boot Data JPA

    • - Para ajudar a trabalhar com a manipulação de dados no projeto.
